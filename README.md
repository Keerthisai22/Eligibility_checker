# Voting Eligibility Checker (Python)

A simple Python program that checks whether a person is eligible to vote based on age and citizenship.

## Features
- Takes real-time user input
- Validates age and citizenship
- Displays eligibility result

## Technologies Used
- Python 3

## How to Run
1. Clone or download the repository
2. Run the program:
   python voting_eligibility_checker.py

## Sample Output

Enter your age: 20

Are you a citizen (Yes/No): yes

You are eligible to vote.

## Concepts Used
- Conditional statements
- Logical operators
- Input handling
